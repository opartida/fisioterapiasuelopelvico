reSmush.it is a module providing an image style effect for the eponymous image optimization API at https://www.resmush.it/

Drupal project page is at https://www.drupal.org/project/resmushit
Installation, demo and tips can be found at http://www.vacilando.org/article/resmushit
If you encounter a problem or need support, see https://www.drupal.org/project/issues/resmushit
